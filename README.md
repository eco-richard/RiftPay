# Welcome to RiftPay!

RiftPay is a website inspired by [SplitWise](https://splitwise.com/). It allows users to share expenses with friends and family. Users can create transactions, organize expenses, track balances, and pay their friends back. 

[Click here to view RiftPay's live site.](https://riftpay.onrender.com)
